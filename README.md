# HR Analytics Dashboard

An end-to-end HR Analytics Dashboard built using SQL and Power BI.

This project demonstrates the complete Business Intelligence workflow, from database design to an interactive dashboard that provides meaningful HR insights.

## Project Overview

This project was built completely from scratch and includes:

- SQL Database Design
- Database Creation & Data Insertion
- Data Cleaning using Power Query
- Data Modeling
- DAX Measures & KPIs
- Interactive Dashboard Development
- Custom KPI Cards using HTML & CSS

## Database Structure

The SQL database consists of 7 interconnected tables:

- Employees
- Departments
- Jobs
- Attendance
- Leave Requests
- Performance
- Training

## Dashboard Pages

### Executive Overview
